# Medifolio
Built a tablet-style hardware device using Raspberry Pi, touchscreen,
and biometric fingerprint sensor for secure and quick access to patient
records.
Enables doctors to access critical medical history immediately during
emergency situations, helping save time and improve decision-making.
# Admin Panel
1. http://[RASPEBERRY_PI_IP_ADDRESS]:5000/  e.g. http://10.32.241.180:5000/

## to add admin & delete all data
2. http://[RASPEBERRY_PI_IP_ADDRESS]:5000/hidden/add-admin  e.g. http://10.32.241.180:5000/hidden/add-admin
